# PROMPTED

A Linux terminal emulator written in Javascript

## Capabilities
### What it can do
+ evaluate commands
+ simulate a basic file structure
+ give some errors
+ basic autocomplete

### What it can't do(yet)
+ pipe
+ exactly replicate a Linux terminal

It's written in Javascript, so just calm down. It's not perfect.
